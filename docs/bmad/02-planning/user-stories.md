# True Valence Mapper - User Stories
*BMAD Phase 2: Planning*
*Product Manager: John*

## Epic 1: ProActive Coach Onboarding

### US1.1: Verify ProActive Certification
**As a** ProActive Certified Coach
**I want to** verify my certification status
**So that** I can access the exclusive ProActive tools

**Acceptance Criteria:**
- Coach email checked against manual allowlist
- System grants access to authorized pilot coaches
- Unauthorized users see "Pilot Access Only" message

**Priority:** P0
**Story Points:** 5

---

### US1.2: Create Coach Profile
**As a** ProActive Certified Coach
**I want to** set up my professional profile
**So that** my clients can identify me easily

**Acceptance Criteria:**
- Profile includes name, photo, credentials
- ProActive certification badge displayed
- Specialization areas selectable
- Contact information (coach-controlled visibility)
- Bio section with ProActive approach

**Priority:** P0
**Story Points:** 3

---

## Epic 2: Relationship Mapping Core

### US2.1: Create Relationship Map
**As a** ProActive Certified Coach
**I want to** create a new relationship map for my client
**So that** we can visualize their professional network

**Acceptance Criteria:**
- One-click new map creation
- Client name and context captured
- Session date automatically recorded
- ProActive framework pre-loaded
- Map saves automatically

**Priority:** P0
**Story Points:** 5

---

### US2.2: Add Relationships
**As a** ProActive Certified Coach
**I want to** add relationships to the map
**So that** we can build a complete picture

**Acceptance Criteria:**
- Add relationship with name and role
- Select from ProActive relationship categories
- Position automatically calculated
- Bulk add option for efficiency
- Duplicate detection

**Priority:** P0
**Story Points:** 8

---

### US2.3: Assess Relationship Valence
**As a** ProActive Certified Coach
**I want to** assess each relationship's quality
**So that** we can identify areas for improvement

**Acceptance Criteria:**
- 5-dimension ProActive assessment
- Slider interface for each dimension (-5 to +5)

- Composite score auto-calculated
- ProActive guidance text displayed

**Priority:** P0
**Story Points:** 13
