# True Valence Mapper

## Project Overview
A workplace relationship mapping and assessment tool for professional coaches and their clients.

## Development Methodology
Using BMAD-METHOD v6-alpha for structured development

## Project Type
- **Classification**: Greenfield (starting from scratch)
- **Track**: BMad Method
- **Current Phase**: Phase 1 - Analysis

## Core Concept
True Valence Mapper helps individuals visualize and assess their workplace relationships, measuring both positive and negative relationship strengths to facilitate intentional relationship improvement through coaching.

## Target Users
- Professional coaches (primary)
- Coaching clients/coachees (secondary)
- HR/Learning & Development teams (tertiary)

## Key Features (Planned)
- Visual relationship network mapping
- Multi-dimensional valence assessment
- Progress tracking over time
- Coach-client collaboration tools
- Action planning integration
